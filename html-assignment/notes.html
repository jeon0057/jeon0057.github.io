<img>
The image tag is a non closing tag that adds an image to a webpage. This requires a source (src) attribute to specify where to find the image. If the content of the image is relevant to the viewer, then alternate text describing the image is also required in case the viewer is unable to see the image for various reasons. This alternate text is also useful for Search Engine Optimization.
<a></a>
The anchor element allows us to add links to our webpage that can lead to anywhere on our web server or the web using an href attribute. The href attribute contains the actual link that tells the computer where to go and the content between the anchor tags is the text that is displayed as the link.
<ul></ul>
The unordered list tag creates a list on a webpage with list items with no particular order.
<blockquote></blockquote>
A blockquote element inserts a quote from another source as its own large standalone content. It requires a citation tag inside of it citing the source of the quote.
<abbr></abbr>
This tag adds an abbreviation to your webpage. The content between the opening and closing tags is the abbreviation that is displayed. The content inside the title attribute in the opening tag is the expanded text which is displayed with a mouse hover.
